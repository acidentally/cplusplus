<I> Khái niệm - Phân loại - Danh pháp
	1, Khái niệm:
		-Là những hchc có nhóm Cacboxyl (-COH) liên kết trực tiếp với H hoặc gốc Hiđrocacbon
			*Cacboxyl là sự kết hợp giữa cacbonyl và hidroxyl (-CO và -H)
		VD: HCOOH (axit fomic) - fomic ~ fomica - kiến
			*COOH ~ CO2H
			CH3COOH (axit axetic)
		CTTQ : Cn H2n+1 COOH (n >= 0)
		hoặc Cx H2x O2 (x >= 1)

	2, Phân loại ~ 2 loại : dựa vào gốc HC (no, không no, thơm)
							dựa vào số nhóm COOH (đơn chức, đa chức)

	3, Danh pháp :
		a) Tên IUPAC (Danh pháp thay thế)
			Axit + tên HC  oic
			VD : HCOOH : axit metanoic
				 CH3 - COOH : axit etanoic
				 CH2 = CH - COOH : axit propenoic
				 CH3 - CH - COOH : 2-metyl propanoic
				 	   |
 				 	   CH3
 		b) Danh pháp thông thường
 			HCOOH : axit fomic
 			CH3COOH : axit axetic
 			C2H5COOH : axit propinoic
 			CH2 = CH - COOH : axit acrylic
 			CH2 = C - COOH : axit metacrylic 
 				  |
 				  CH3
 			COOH : axit oxalic
 			|
 			COOH

 			COOH - (CH2)4 - COOH : axit adipic
 			CH3 - CH - COOH : axit lactic (axit tạp chức)
 				  |
 				  OH

<III> Tính chất vật lý (SGK)
	- Axit có liên kết Hiđro liên phân tử nên nhiệt độ sôi và nhiệt độ nóng chảy cao
	- Axit có liên kết Hiđro với nước nên tan tốt trong nước
<IV> Tính chất hóa học :
	1, Tính axit : +) Quỳ hóa đỏ : RCOOH <==> RCOO- + H+
				   +) tác dụng KL trước H -> H2 ^
				   +) tác dụng oxit bazơ, dung dịch bazơ
				   +) tác dụng muối -> muối kết tủa hoặc axit mới dễ bay hơi
	2, Phản ứng thế nhóm -OH (Este hóa) :

		RCOOH + R'OH ---> R - C - O - R' + H2O
						     ||
						      O
		Phản ứng giữa axit + ancol -> este hóa
		Đặc điểm : phản ứng thuận nghịch (có xúc tác H2SO4đ)
<V> Điều chế / Ứng dụng
	1, Điều chế :
		- Lên men giấm :
			C2H5OH + O2 ------men giấm------> CH3COOH + H2O
		- Từ andehit :
			RCHO + 1/2O2 ----Mn2+----> RCOOH
		- Oxi hóa ankan :
			CH3 - CH2 - CH2 - CH3 + 5/2O2 ---xt---> 2CH3COOH + H2O
			R - CH2 - CH2 - R' + 5/2O2 ---xt---> RCOOH + R'COOH + H2o
		- Điều chế riêng trong công nghiệp :
			CH3OH + CO --xt--t độ--> CH3COOH
